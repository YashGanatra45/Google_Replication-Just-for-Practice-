# 🔍 Google Homepage Clone

A pixel-perfect recreation of the **Google.com** homepage built with pure **HTML** and **CSS** — no JavaScript frameworks, no libraries, just clean markup and styling.

---

## 📸 Preview

> The clone replicates the classic Google homepage layout including the logo, search bar, buttons, navigation, and footer.

---

## 🚀 Features

- ✅ Google-style **colored logo** (4 brand colors)
- ✅ **Search bar** with search icon, voice icon, and camera (search by image) icon
- ✅ Hover & focus **shadow effect** on the search bar
- ✅ **"Google Search"** and **"I'm Feeling Lucky"** buttons
- ✅ Top-right **navigation links** (About, Store, Gmail, Images, Apps grid, Sign In)
- ✅ **Language options** row (localized for India)
- ✅ Realistic **footer** with country label and links
- ✅ Pure HTML & CSS — **zero dependencies**

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Page structure & semantic markup |
| CSS3 | Layout, typography, transitions, hover effects |
| SVG | Icons (search, voice, camera, apps grid) |

## ▶️ Getting Started

No setup required. Just open the file in a browser:

```bash
# Clone the repository
git clone https://github.com/your-username/google-homepage-clone.git

# Navigate into the project folder
cd google-homepage-clone

# Open in your browser
open index.html
```

Or simply double-click `index.html` to open it in your default browser.

---

## 🎨 Design Highlights

- **Logo** — Built with pure HTML/CSS using Google's official brand colors: `#4285F4` (blue), `#EA4335` (red), `#FBBC05` (yellow), `#34A853` (green)
- **Search Bar** — Rounded border with a smooth `box-shadow` transition on hover/focus, matching Google's interaction pattern
- **Buttons** — Light grey background with subtle border and shadow on hover
- **Footer** — Two-section layout mirroring Google's actual footer structure

---

## 📌 What's Not Included

- Search functionality (no backend/API)
- JavaScript interactions (autocomplete, suggestions)
- Responsive/mobile breakpoints (desktop layout only)
- Dark mode
